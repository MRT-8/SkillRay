# File Organizer Skill

This skill helps organize files in a directory by type.

## Usage

Run `organize` to sort files into subdirectories by extension.

## Supported Extensions

- Images: .jpg, .png, .gif, .svg
- Documents: .pdf, .docx, .txt
- Code: .py, .js, .ts

## How it works

The skill reads the current directory listing, creates subdirectories for each file type,
and moves files into the appropriate subdirectory.

## Requirements

- Python 3.10+
- No external dependencies
